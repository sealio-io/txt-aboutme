In angular when defining a selector I learned that it doesnt have to be an element. It can be an attribute, class, or Id! I learned that when outputting data you can use the follow 2 techniques. String Interpolation - {{data}} and Property Binding - [property]="data". When you want to react to a user's click or anything the user interacts with you can use Event Binding - (event)="expression". A combination of both property binding and event binding is known as TWO-WAY-BINDING - [(ngmodel)]="data". Also in typescript when creating a variable your able to define what data type it is...reminds me of C++. You can do this by typing - varname : (number,string, etc.).
